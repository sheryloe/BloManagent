공개 블로그를 점수와 근거로 진단하는 서비스형 워크스페이스

BloManagent는 URL만 받아도 공개 블로그를 수집하고, verified post만 기준으로 품질 점수와 개선 포인트를 보여주도록 설계된 분석 서비스입니다. 핵심은 “예쁘게 보이기”가 아니라 “왜 이 점수가 나왔는지 설명 가능한가”입니다.

저장소 보기 도움말

Input

메인 URL / 게시글 URL

Rule

verified post only

Output

S ~ F 등급 + 개선 액션

Discovery

RSS, sitemap, wp-json, HTML fallback을 순서대로 시도합니다.

Verification

카테고리, 태그, 아카이브를 제외하고 실제 게시글만 집계합니다.

Scoring

headline, readability, value, originality, search fit을 조합합니다.

Recommendation

AI는 점수 계산이 아니라 설명 보강과 개선 액션 생성에 사용합니다.

Pipeline

분석은 수집보다 정제가 더 중요합니다

01 Collect

공개 영역만 수집합니다.

02 Verify

진짜 게시글만 남깁니다.

03 Score

알고리즘 기준으로 등급을 계산합니다.

04 Explain

근거와 개선 포인트를 리포트로 보여줍니다.

Product

공유 가능한 리포트

분석 결과를 링크나 파일로 쉽게 공유할 수 있어야 서비스 가치가 커집니다.

Data

기간별 비교

같은 블로그를 주기적으로 비교해 개선 추세를 볼 수 있게 해야 합니다.

Ops

비동기 분석 큐

긴 수집과 분석을 큐 기반으로 돌리면 서비스 품질과 안정성이 좋아집니다.
