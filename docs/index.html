메인 URL만으로 시작하는
공개 블로그 수집과 분석

BloManagent는 티스토리, 네이버 블로그, Blogger, WordPress 같은 공개 블로그를 등록하고 수집한 뒤, 운영 지표와 AI 기반 보완 포인트를 한 화면에서 보려는 사람을 위한 오픈소스 워크스페이스입니다.

대시보드에는 무엇이 보이나요?

블로그별 EBI, 최근 추천 액션, 마지막 수집 시각, 최근 실행 기록을 한 번에 확인할 수 있습니다.

01

URL-only 등록

블로그 메인 주소만으로 시작하고, RSS URL 같은 값은 필요한 경우에만 고급 옵션으로 넣습니다.

02

플랫폼별 공개 수집

네이버 블로그 RSS, 티스토리 RSS/사이트맵, WordPress wp-json 같은 공개 구조를 우선 활용합니다.

03

보완 포인트 정리

단순 요약이 아니라 블로그별 상태, 개선 우선순위, 다음 액션을 대시보드와 리포트에 쌓습니다.

Workflow

실제 사용 흐름

1차 목표는 어렵지 않은 시작점입니다. 공개 블로그 URL을 넣고, 수집하고, 분석하고, 다음 글 방향을 정리하는 과정이 한 번에 이어져야 합니다.

1. 블로그 등록

티스토리, 네이버 블로그, Blogger, WordPress 메인 URL을 등록합니다.

2. 포스트 수집

공개 RSS, sitemap, wp-json, 메인 링크를 차례로 확인해 포스트 URL을 발견합니다.

3. 분석 실행

최근 7일, 30일, 신규 글만 등 범위를 정해 분석을 돌리고 결과를 누적 저장합니다.

4. 보완 액션 확인

블로그별 점수 변화와 추천 액션을 보고 다음 글 기획과 운영 우선순위를 잡습니다.

Safe use

프로젝트 기본 원칙

공개 페이지 중심

로그인이 필요한 영역이나 비공개 글을 우회해서 수집하지 않고, 공개 URL 기준으로만 접근합니다.

오픈소스 워크스페이스

상용 SaaS보다 개인 또는 팀이 직접 실행하는 분석 도구에 가깝게 설계했습니다.

약관과 robots 확인 권장

공개 페이지라도 자동 접근 정책은 서비스별로 다를 수 있으므로 배포/호스팅 전에 확인하는 쪽이 안전합니다.

FAQ

자주 묻는 질문

BloManagent는 무엇을 하나요?

공개 블로그 메인 URL을 등록하면 수집과 분석을 묶어서 블로그별 운영 상태와 보완 포인트를 보여 줍니다.

도움말은 어디서 보나요?

도움말 / 매뉴얼 페이지에서 등록 예시, 수집 방식, 안전하게 사용하는 기준을 확인할 수 있습니다.

저장소 주소는 어디인가요?
