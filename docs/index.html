게시글별 품질 점수를 확인합니다

BloManagent는 공개 블로그 URL을 기준으로 글을 수집하고, 게시글마다 설명 가능한 `qualityScore`와 개선 포인트를 계산하는 로컬 중심 오픈소스 도구입니다.

  • 메인 URL 등록
  • RSS / sitemap / wp-json 수집
  • algorithm 기본 분석
01

URL-only 등록

메인 URL만 입력하면 공개 엔드포인트를 자동으로 탐색합니다.

02

알고리즘 기본 분석

API 키 없이도 게시글 점수와 개선 제안을 바로 계산할 수 있습니다.

03

AI는 선택형 보강

OpenAI, Google, Ollama는 요약 문장 보강 용도로만 선택할 수 있습니다.

Workflow

사용 흐름

수집과 분석, 우선순위 확인까지 한 번에 연결되는 구조입니다.

1. 메인 URL 등록

Tistory, Blogger, WordPress 메인 URL을 넣습니다.

2. 공개 글 수집

RSS, sitemap, wp-json, 메인 링크를 순서대로 확인합니다.

3. 게시글 분석

글마다 qualityScore와 개선 제안을 계산합니다.

4. 우선순위 확인

대시보드에서 낮은 점수 글과 반복 이슈를 먼저 봅니다.

Policy

기본 원칙

공개 페이지만 대상

로그인, 비공개 글, 권한 우회가 필요한 영역은 수집하지 않습니다.

네이버는 기본 비활성

정책 리스크 때문에 `allowNaverPublicCrawl=false`를 기본값으로 둡니다.

로컬 중심 사용

다중 사용자 SaaS보다는 직접 실행하는 오픈소스 워크플로우를 목표로 합니다.
