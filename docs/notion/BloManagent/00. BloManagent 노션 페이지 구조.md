# BloManagent 노션 페이지 구조

이 폴더는 노션에 바로 옮겨 담을 수 있게 구성한 문서 세트입니다.  
상위 페이지 이름은 **BloManagent** 로 만들고, 그 아래에 다음 4개 하위 페이지를 넣으면 됩니다.

## 권장 페이지 트리

- Step 1. BloManagent 기획 배경과 로컬 퍼스트 아키텍처 설계
- Step 2. SQLite 스키마와 블로그 수집기 구현 과정
- Step 3. AI 분석 파이프라인과 대시보드 구현 과정
- Step 4. 테스트, 빌드, Windows 실행, 향후 확장 전략

## 노션 배치 팁

- 상위 페이지 제목: `BloManagent | 로컬 퍼스트 블로그 분석 대시보드 제작기`
- 상위 페이지 설명 첫 문단:
  `티스토리, 네이버 블로그, Blogger를 메인 URL 기준으로 수집하고 주간 분석 이력을 저장하는 로컬 퍼스트 블로그 분석 대시보드 개발 기록입니다.`
- 각 Step 페이지는 그대로 복사해도 되고, Markdown import로 넣어도 됩니다.
- 티스토리 블로그 형태를 고려해 각 문서는 제목, 메타 설명, 목차, 본문, 마무리 구조로 작성했습니다.
